Project Management Notes

Nov 20, 2021 3 min read Learning PM

Recently, I enrolled the Project Management course in Coursera. The course is delivered by Google.

I found this is interesting, because usually we have some basic knowledge how to manage a small team, and how to manage a simple project.

For example, plan a trip, upgrade the gararge, etc.

After going through the first course, I found this course can really help to correct my mindset and change the way how to communicate with other people.

Is project management necessary?

Till last year, I am in charge of the release, I realize that project management is really important. A project really needs a good plan and also after the major work is done, doing some follow-ups, etc.

We have a release happening every several months. First several releases is OK since we have less customers, and did not get a lot of support. So the Google’s online course “Foundations of Project Management” introduces what is project management, what is project manager, and what is the role of project managers in different organziations.

Some of the senior manages in my organization will play the role of project managers, plan the project, communicate with teams and stakeholders.

As I was promoted to a new role, I was starting to manage some tasks and keep eys on them.

Course Note “Foundations of Project Management”

After finish the Course one, I have questions to ask myself.

What is the projet managers’ role in my team

If our team will hire a project manager, how these skills which are discussed in the course can help our team. Basically, before the task is assigned to the team member, we can have project manager initialize the project and track the progress of the project.

Our work is providing service to other teams, and the commitment is very important. So to have a project manager will be helpful. Actually we have several program managers, but they just focus on something more high level.

What if I am the project manager

If I am a project manager, I would like to show my impact and make the project.

How to make my personal skill more professional

Most people already have some project management skill or the sense how to manage a project.

Just these skills sometimes is a personal ability, there are better ways we can learn and tune our personal skill to be the professional ability.

Project management lifecycle and methodologies

  • Split project into different phases
  • Closing project is important
  • Agile
  • Waterfall
  • Lean and Six Sigma

As a lead or manager, tool is necessary.

  • master Excel
  • Email Writing
  • Todo list
  • Issue tracking tool, like Jira
  • Graph basis to express data
  • Python
  • AHK
  • etc.
